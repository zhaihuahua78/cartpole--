\section{问题分析}

\subsection{Cart Pole系统与强化学习环境}

Cart Pole（小车-倒立摆）是自动控制原理中经典的非线性系统控制案例。系统由一辆可在轨道上水平移动的小车和铰接于其上的摆杆组成。控制目标是施加水平力使摆杆始终保持直立。

本实验使用符合任务描述的自定义CartPole环境，基于OpenAI Gymnasium的物理引擎实现，修改了以下参数：
\begin{itemize}[nosep]
    \item \textbf{终止角度}：由CartPole-v1默认的12°改为\textbf{15°}（0.2618 rad），与任务描述一致；
    \item \textbf{失败惩罚}：当摆杆倾斜超过15°或小车超出轨道时，给予\textbf{-10}的惩罚，与任务描述"获得相应的负向惩罚"一致；
    \item 每步存活奖励+1、小车边界2.4m、最大步数500保持不变。
\end{itemize}

状态空间为4维连续向量，定义如表\ref{tab:state}。

\subsection{算法选择及基本原理}

本实验选用DQN（值函数方法）和PPO（策略梯度方法）两种代表性算法进行对比研究。

\textbf{DQN（Deep Q-Network）}使用深度神经网络近似最优Q函数，损失函数为：
\begin{equation}
L(\theta) = \mathbb{E}\left[\left(r + \gamma \max_{a'} Q(s', a'; \theta^-) - Q(s, a; \theta)\right)^2\right]
\end{equation}
核心设计包括经验回放缓冲区（100,000容量）、目标网络（每1,000步更新）和\(\varepsilon\)-greedy探索。超参数：学习率\(1\times10^{-4}\)，batch 32，\(\gamma=0.99\)。

\textbf{PPO（Proximal Policy Optimization）}直接优化随机策略，通过裁剪机制稳定更新：
\begin{equation}
L^{\text{CLIP}} = \mathbb{E}\left[\min(r_t \hat{A}_t, \text{clip}(r_t, 1-\epsilon, 1+\epsilon)\hat{A}_t)\right]
\end{equation}
超参数：学习率\(3\times10^{-4}\)，n\_steps 2,048，clip\_range 0.2，GAE \(\lambda=0.95\)。

\subsection{奖励函数设计}

奖励函数设计是本次实验的核心环节。根据任务描述要求，奖励规则如下：
\begin{itemize}[nosep]
    \item 每步成功保持平衡：\(r = +1\)（正向奖励）
    \item 失败（\(\theta>15^\circ\)或\(x>2.4\)m）：\(r = -10\)（负向惩罚）
    \item 成功坚持500步：\(r = +1\)（与每步一致，不做额外加分）
\end{itemize}

\section{实验过程}

\subsection{环境构建与参数设置}

使用自定义CartPole环境（自实现物理引擎，与Gymnasium CartPole-v1一致）。PPO训练100,000步，DQN训练300,000步。随机种子42，CPU设备。

\subsection{PPO训练过程}

PPO在100,000步内完美求解自定义环境（图\ref{fig:ppo}）。训练过程：
- 探索期（0--200回合）：奖励10--40，与随机策略相当
- 收敛期（200--280回合）：奖励快速攀升至500
- 稳定求解期（280回合后）：持续满分500

\begin{figure}[H]
\centering
\includegraphics[width=0.85\textwidth]{results_task/training_curves.png}
\caption{PPO（右）与DQN（左）在自定义环境中的训练曲线。PPO稳定收敛至满分，DQN经历了完整的学习-遗忘过程。}
\label{fig:ppo}
\end{figure}

训练汇总：总回合434，最后100回合均值\textbf{500.0}，最高奖励500。20回合评估全部满分（500.0±0.0）。

\subsection{DQN训练过程}

DQN训练300,000步（共7,711回合），呈现三阶段模式（图\ref{fig:dqn}）：

\begin{itemize}[nosep]
    \item 探索期（0--1,200回合）：均值10--20，与随机策略无异
    \item 学习上升期（1,200--2,200回合）：均值从20攀升至峰值389（第2,200回合）
    \item 灾难性遗忘期（2,200回合后）：性能持续衰退至均值2--5，评估仅2.0
\end{itemize}

DQN未达到满分，满分率0\%。失败惩罚（-10）加速了遗忘过程：策略变差后频繁获得负奖励，导致Q值快速发散。

\subsection{传统控制对比}

\begin{itemize}[nosep]
    \item \textbf{随机策略}：平均12.3步
    \item \textbf{规则控制器}：基于角度阈值反馈，平均215.8步
    \item \textbf{LQR控制器}：线性化最优控制，平均44.8步
\end{itemize}

\subsection{灾难性遗忘分析}

DQN在自定义环境中的灾难性遗忘比标准CartPole-v1更加严重（评估均值2.0 vs 10.2），原因在于失败惩罚（-10）形成了恶性循环：
\begin{enumerate}[nosep]
    \item DQN学到一定策略后Q值估计开始发散
    \item 策略退化导致频繁触碰15°边界
    \item -10惩罚压低Q值估计，进一步恶化策略
    \item 回放缓冲区中高质量经验被低质量经验覆盖
    \item 最终策略完全崩溃
\end{enumerate}

\subsection{PPO与DQN根本差异}

PPO成功而DQN失败的核心原因：
\begin{itemize}[nosep]
    \item PPO直接优化策略网络，不依赖Q值自举估计，避免误差累积
    \item PPO的裁剪机制限制单步更新幅度，保证学习稳定性
    \item DQN的\(\varepsilon\)-greedy探索在负惩罚环境下效率极低
\end{itemize}

\section{总结体会}

本实验严格按照任务描述实现了自定义CartPole环境（15°终止角、-10失败惩罚），并对比了五种控制方法。

\textbf{核心结论：}
(1) PPO完美解决自定义环境（100\%达标率，500满分），展示了策略梯度方法的卓越稳定性和样本效率；
(2) DQN出现灾难性遗忘（评估均值2.0），在失败惩罚机制下遗忘速度更快、程度更深；
(3) 规则控制器（215.8分）优于LQR（44.8分），对强非线性系统启发式设计优于线性化最优控制。

\textbf{对任务描述的验证：}
实验证明15°终止角和-10惩罚的设计是合理的——PPO能够学会在该约束下完美控制，而DQN的失败则凸显了算法选择对控制性能的决定性影响。

\textbf{改进方向：}引入Double DQN、优先经验回放等改进DQN；尝试SAC、A2C等更多算法；在参数扰动下测试泛化性。
